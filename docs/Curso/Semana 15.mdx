# Semana 15

## Tutorial de Layouts en Android Studio

## 1. ConstraintLayout Tutorial

### Paso 1: Configuración Inicial
```xml
<?xml version="1.0" encoding="utf-8"?>
<androidx.constraintlayout.widget.ConstraintLayout
    xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto"
    android:layout_width="match_parent"
    android:layout_height="match_parent">
```

### Paso 2: Crear un Formulario de Login
```xml
    <!-- Logo en la parte superior -->
    <ImageView
        android:id="@+id/logoImage"
        android:layout_width="120dp"
        android:layout_height="120dp"
        android:layout_marginTop="48dp"
        app:layout_constraintTop_toTopOf="parent"
        app:layout_constraintStart_toStartOf="parent"
        app:layout_constraintEnd_toEndOf="parent"
        android:src="@drawable/logo"/>

    <!-- Campo de Email -->
    <EditText
        android:id="@+id/emailInput"
        android:layout_width="0dp"
        android:layout_height="wrap_content"
        android:layout_marginTop="32dp"
        android:layout_marginHorizontal="24dp"
        android:hint="Email"
        android:inputType="textEmailAddress"
        app:layout_constraintTop_toBottomOf="@id/logoImage"
        app:layout_constraintStart_toStartOf="parent"
        app:layout_constraintEnd_toEndOf="parent"/>

    <!-- Campo de Contraseña -->
    <EditText
        android:id="@+id/passwordInput"
        android:layout_width="0dp"
        android:layout_height="wrap_content"
        android:layout_marginTop="16dp"
        android:hint="Contraseña"
        android:inputType="textPassword"
        app:layout_constraintTop_toBottomOf="@id/emailInput"
        app:layout_constraintStart_toStartOf="@id/emailInput"
        app:layout_constraintEnd_toEndOf="@id/emailInput"/>

    <!-- Botón de Login -->
    <Button
        android:id="@+id/loginButton"
        android:layout_width="0dp"
        android:layout_height="wrap_content"
        android:layout_marginTop="24dp"
        android:text="Iniciar Sesión"
        app:layout_constraintTop_toBottomOf="@id/passwordInput"
        app:layout_constraintStart_toStartOf="@id/passwordInput"
        app:layout_constraintEnd_toEndOf="@id/passwordInput"/>
</androidx.constraintlayout.widget.ConstraintLayout>
```

### Conceptos Clave del ConstraintLayout:
1. Restricciones obligatorias: Cada vista necesita restricciones horizontal y vertical
2. Márgenes y padding
3. Guías y barreras
4. Cadenas de elementos
5. Dimensiones relativas (porcentaje)



## 2. LinearLayout Tutorial

### Paso 1: Configuración Inicial
```xml
<?xml version="1.0" encoding="utf-8"?>
<LinearLayout
    xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical"
    android:padding="16dp">
```

### Paso 2: Crear una Lista de Contactos
```xml
    <!-- Encabezado -->
    <TextView
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:text="Lista de Contactos"
        android:textSize="24sp"
        android:gravity="center"
        android:layout_marginBottom="16dp"/>

    <!-- Contacto 1 -->
    <LinearLayout
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:orientation="horizontal"
        android:layout_marginBottom="8dp">

        <ImageView
            android:layout_width="48dp"
            android:layout_height="48dp"
            android:src="@drawable/contact_icon"/>

        <LinearLayout
            android:layout_width="0dp"
            android:layout_height="wrap_content"
            android:layout_weight="1"
            android:orientation="vertical"
            android:layout_marginStart="8dp">

            <TextView
                android:layout_width="wrap_content"
                android:layout_height="wrap_content"
                android:text="Juan Pérez"
                android:textStyle="bold"/>

            <TextView
                android:layout_width="wrap_content"
                android:layout_height="wrap_content"
                android:text="+1 234 567 890"/>
        </LinearLayout>

        <Button
            android:layout_width="wrap_content"
            android:layout_height="wrap_content"
            android:text="Llamar"/>
    </LinearLayout>
</LinearLayout>
```

### Conceptos Clave del LinearLayout:
1. Orientación (vertical/horizontal)
2. Layout_weight para distribución proporcional
3. Anidamiento de layouts
4. Gravity vs Layout_gravity
5. Márgenes y padding



## 3. RelativeLayout Tutorial

### Paso 1: Configuración Inicial
```xml
<?xml version="1.0" encoding="utf-8"?>
<RelativeLayout
    xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:padding="16dp">
```

### Paso 2: Crear una Pantalla de Perfil
```xml
    <!-- Imagen de perfil -->
    <ImageView
        android:id="@+id/profileImage"
        android:layout_width="120dp"
        android:layout_height="120dp"
        android:layout_centerHorizontal="true"
        android:layout_marginTop="32dp"
        android:src="@drawable/profile_pic"/>

    <!-- Nombre de usuario -->
    <TextView
        android:id="@+id/userName"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_below="@id/profileImage"
        android:layout_centerHorizontal="true"
        android:layout_marginTop="16dp"
        android:text="María González"
        android:textSize="24sp"
        android:textStyle="bold"/>

    <!-- Bio -->
    <TextView
        android:id="@+id/userBio"
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:layout_below="@id/userName"
        android:layout_marginTop="8dp"
        android:gravity="center"
        android:text="Desarrolladora Android | Amante del café"/>

    <!-- Botones de acción -->
    <Button
        android:id="@+id/editProfileButton"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_below="@id/userBio"
        android:layout_centerHorizontal="true"
        android:layout_marginTop="24dp"
        android:text="Editar Perfil"/>

    <!-- Estadísticas -->
    <LinearLayout
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:layout_below="@id/editProfileButton"
        android:layout_marginTop="32dp"
        android:orientation="horizontal"
        android:weightSum="3">

        <TextView
            android:layout_width="0dp"
            android:layout_height="wrap_content"
            android:layout_weight="1"
            android:gravity="center"
            android:text="500\nSeguidores"/>

        <TextView
            android:layout_width="0dp"
            android:layout_height="wrap_content"
            android:layout_weight="1"
            android:gravity="center"
            android:text="300\nSiguiendo"/>

        <TextView
            android:layout_width="0dp"
            android:layout_height="wrap_content"
            android:layout_weight="1"
            android:gravity="center"
            android:text="50\nPublicaciones"/>
    </LinearLayout>
</RelativeLayout>
```

### Conceptos Clave del RelativeLayout:
1. Posicionamiento relativo a otros elementos
2. Alineación con el padre
3. Márgenes y padding
4. Combinación con otros layouts
5. Centrado de elementos

## Consejos Generales para Todos los Layouts

### 1. Optimización de Rendimiento
- Evitar anidamientos profundos
- Usar herramientas de análisis de jerarquía
- Preferir ConstraintLayout para layouts complejos

### 2. Buenas Prácticas
- Usar dimensiones y strings como recursos
- Implementar soporte para diferentes tamaños de pantalla

### 3. Depuración
- Usar Layout Validation
- Validar constraints

### 4. Consideraciones de Diseño
- Material Design guidelines
- Densidad de píxeles (dp vs px)
