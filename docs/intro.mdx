# Introducción

<div class="flex justify-center"> <img src="https://images.unsplash.com/photo-1630442923896-244dd3717b35?q=80&w=2070&auto=format&fit=crop&ixlib=rb-4.0.3&ixid=M3wxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8fA%3D%3D" alt="Programación de Aplicaciones Móviles II" class="max-w-full h-auto" /> </div>

Bienvenidos al submódulo de Programación de Aplicaciones Móviles II, parte del programa Técnico Laboral como Asistente en Desarrollo de Software ofrecido por CESDE.

Este submódulo está diseñado para sumergirte en el mundo del desarrollo de aplicaciones móviles, centrándose en dos unidades principales:

1. Momento 1: Componentes de desarrollo
    -   Creación de aplicaciones móviles con SQLite CRUD y RecyclerView
2. Momento 2: Metodologías de desarrollo en aplicaciones móviles
    -   Introducción a Firebase, Firebase Firestore
2. Momento 3: Proyecto Integrador
    -   Firebase Authentication y consumo de Webservices

A lo largo de 18 semanas, explorarás temas como la creación de aplicaciones móviles utilizando SQLite, la integración con la base de datos NoSQL de Firebase, la implementación de autenticación de usuarios y el consumo de servicios web.

Un aspecto destacado del submódulo es el Proyecto Integrador, en el que tendrás seguimientos periódicos y una sustentación final para demostrar tus habilidades en el desarrollo de aplicaciones móviles.
